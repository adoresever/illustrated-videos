# Attributed reference method

Source: 季白羽 ([@vbjby3](https://x.com/vbjby3/status/2076530524110369070)), “我用 Codex + Remotion，做了一条唐朝纸片分层动画”, X post 2076530524110369070, published 2026-07-13.

The archived post contained a long-form tutorial and six screenshots; no downloadable final video was found in the accessible post. Treat this as an article-derived production method, not a frame-by-frame analysis of the final animation.

The reusable method is:

1. Decide narration and shots before generating assets.
2. Separate background, distant characters, primary subject, and foreground.
3. Generate backgrounds without characters.
4. Generate characters on a removable flat background.
5. Split sprite sheets and remove backgrounds into independent PNGs.
6. Establish static composition and hierarchy before motion.
7. Map primary, secondary, and tertiary roles to different entrance distances and delays.
8. Use occlusion, size difference, small camera movement, captions, music, and sound effects for depth and rhythm.
9. Preview in Remotion Studio and use FFmpeg for final QA.

Quality observation from the archived screenshots: the strongest frames use several asymmetrically scaled subjects, active foreground/background occlusion, torn-paper decoration, a clear value contrast behind the primary, light captions, and sound-oriented entrance timing. A technically layered video can still feel flat when it has only one centered subject, repeated plates, equal object scale, or several opaque text cards. Use the general quality rubric rather than copying this case's Tang characters, red field, or typography.

Do not distribute the source screenshots with this Skill without permission. Preserve attribution when teaching from this case.
